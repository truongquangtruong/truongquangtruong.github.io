## Giới thiệu
Trong quá trình nghiên cứu lập trình của mình, tôi đã bắt đầu với Java vì đây là ngôn ngữ mạnh mẽ và phổ biến. Java không chỉ giúp tôi hiểu về lập trình hướng đối tượng mà còn áp dụng vào các dự án thực tế như ứng dụng di động. Khóa học này là chia sẻ từ những gì tôi học được.

## Lịch sử phát triển của Java
Java được James Gosling phát triển tại Sun Microsystems năm 1995, ban đầu tên "Oak". Nó nhanh chóng trở thành chuẩn cho lập trình đa nền tảng nhờ JVM.

<grok-card data-id="f781d1" data-type="image_card"  data-arg-size="LARGE" ></grok-card>


Tôi nhận thấy lịch sử Java gắn liền với sự phát triển của web và mobile.

## Ưu điểm và ứng dụng thực tế
Ưu điểm: Độc lập nền tảng, bảo mật cao, thư viện phong phú. Ứng dụng: Android, Spring cho web.

<grok-card data-id="f0ef36" data-type="image_card"  data-arg-size="LARGE" ></grok-card>


Từ nghiên cứu, tôi thấy Java lý tưởng cho enterprise.

## Cài đặt môi trường và chương trình đầu tiên
Cài JDK từ oracle.com. Ví dụ code:

```java
public class HelloJava {
    public static void main(String[] args) {
        System.out.println("Xin chào Java từ nghiên cứu của tôi!");
    }
}
